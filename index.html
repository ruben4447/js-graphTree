Controls

Drag and drop to move nodes

Left-click on a node to start a connection. Left-click on another node to connect them, or anywhere else to stop connection.

Right-click on nodes to re-name them, or on connections to change their value, or on the white background to create a new node

Press 'delete' key over node/connection to remove it. Press 's' over a node to start Dikstra's algorithm (step-through), or press 'enter' to run Dijkstra's algorithm to completion
